# HavenIQ

## Buy Smarter. Invest Confidently.

HavenIQ is an AI-powered property investment analysis platform designed to help buyers and investors make informed decisions before purchasing real estate.

---

## Vision

To become the world's most trusted AI property investment advisor.

---

## Mission

Help millions of people avoid bad property investments through transparent, data-driven AI analysis.

---

## Version 1 Goals

- Analyze a property from an address or listing URL
- Generate an AI investment report
- Estimate appreciation potential
- Estimate rental potential
- Explain investment risks
- Provide a Buy / Wait / Avoid recommendation
