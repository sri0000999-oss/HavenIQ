# HavenIQ

## Buy Smarter. Invest Confidently.

HavenIQ is an AI-powered property investment analysis platform that helps homebuyers and real estate investors make informed decisions before purchasing a property.

## Vision

To become the world's most trusted AI property investment advisor.

## Mission

Help millions of people avoid bad property investments through transparent, AI-powered analysis.

## Version 1 Features

- Analyze a property using an address or listing URL
- AI-generated investment report
- Appreciation estimate
- Rental income estimate
- Investment risk analysis
- Buy / Wait / Avoid recommendation

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
